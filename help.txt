0. git init - инициализирует  git 
1. git status - показывает состояние
2. git add . - добавляет все файлы в подготовленное состояние
3. git commit -m "тексте сообщения " - переводит подготовленные файлы в фиксированное состояние 
4. git log  - посмотреть историю
5. git push - отправить изменения
6. git pull - получить изменения
7. git clone url - вытащить новый проект
